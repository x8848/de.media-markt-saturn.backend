## Description

Nest GraphQL API

## Installation

```bash
$ yarn install
```

## Database setup

Install & Run MondoDB

https://www.mongodb.com/docs/manual/administration/install-community/

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
