## Description

Nest MondoDB Apollo GraphQL API

## Installation

```bash
$ yarn
```

## Database setup

Install & Run MondoDB

https://www.mongodb.com/docs/manual/administration/install-community/

## Database seeding

```bash
$ yarn db:seed
```

## Running the app

```bash
# development
$ yarn dev

# debug mode
$ yarn debug

# production mode
$ yarn build
$ yarn start
```
